# Password manager

## How to

Git clone the repository 
`git clone https://github.com/romainlq/password-manager.git`

### Start client

`cd client && npm i && npm run dev`  

### Start server

`cd server && npm i && npm run dev`

## Tech stack

### Client

- ViteJs
- React
- Redux toolkit

### Server

- koa
- postgresql database
